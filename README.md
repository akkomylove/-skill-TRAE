## 🎯 核心功能

### 1. 论文写作技巧

- 标准数学建模论文结构模板
- 摘要、引言、结论等各部分写作指南
- 常用学术短语模式
- 语言表达规范

### 2. 数学建模方法

涵盖中国数学建模竞赛常用的各类模型：

- 优化类：线性规划、非线性规划、整数规划、动态规划
- 预测类：时间序列、灰色预测、神经网络
- 统计类：回归分析、主成分分析、聚类分析
- 评价类：层次分析法、TOPSIS、模糊评价、熵权法
- 图论类：最短路径、最小生成树、最大流
- 微分方程：常微分方程、偏微分方程、SIR模型
- 其他：排队论、博弈论、元启发式算法

### 3. LaTeX公式指南

提供各类数学模型的LaTeX公式模板，包括：

- 优化模型的目标函数与约束条件
- 预测模型公式
- 统计模型公式
- LaTeX基础语法与编辑技巧

### 4. 数据可视化

提供10种常用图表的Python和Matlab双语言代码模板：

- 折线图、柱状图、散点图
- 饼图、热力图、误差棒图
- 直方图、箱线图、3D图、子图布局
- 学术图表设计原则

### 5. 术语词典

按类别整理数学建模常用术语，包含：

- 中文术语
- 英文翻译
- 术语定义
- 12个专业类别

### 6. 参考文献格式

提供4种常用引用格式：

- GB/T 7714-2015（中国国家标准）
- APA格式
- IEEE格式
- MLA格式

## 📚 使用指南

### 快速开始

1. **论文结构规划**：参考 `1.论文写作技巧/论文结构模板.md`
2. **选择模型**：根据题目类型在 `2.数学建模方法/` 中选择合适模型
3. **公式编辑**：使用 `3.LaTeX公式指南/` 中的模板
4. **数据可视化**：在 `4.数据可视化/` 中选择图表模板
5. **术语查询**：在 `5.术语词典/` 中查找专业术语
6. **参考文献**：在 `6.参考文献格式/` 中选择引用格式

### 不同题型建议

| 题目类型    | 推荐章节                | 重点内容            |
| :------ | :------------------ | :-------------- |
| A题（优化类） | 2.数学建模方法/优化类模型.md   | 目标函数、约束条件、求解算法  |
| B题（预测类） | 2.数学建模方法/预测类模型.md   | 时间序列、灰色预测、误差分析  |
| C题（评价类） | 2.数学建模方法/评价类模型.md   | AHP、TOPSIS、权重确定 |
| D题（仿真类） | 2.数学建模方法/微分方程类模型.md | ODE、PDE、系统仿真    |
